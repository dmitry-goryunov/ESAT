# ESAT Preparation Notes

## Format reference

| Feature | ESAT |
|---|---|
| Question type | Multiple choice |
| Calculator | No |
| Time per module | about 40 minutes |
| Questions per module | about 27 |
| Average time per question | about 1.5 minutes |
| Difficulty style | short, fast, deceptive |
| Main skill | rapid recognition of patterns and shortcuts |

Typical Physics questions are concise, algebra-heavy and conceptually deceptive. A full derivation is often too slow.

| Test | Usefulness | Main use |

## ESAT official material rule

Use official ESAT online material from Week 6 onwards only. Specimens with explanations should be fully reviewed. Samples with correct/incorrect only should be treated as pressure tests.

## PAT usage

PAT is for selected concept repair only. Do not use full PAT papers as the main benchmark because the format is longer and less multiple-choice driven.
