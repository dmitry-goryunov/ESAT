# ESAT Preparation Project

This repository organises preparation for:
- ESAT Mathematics 1
- ESAT Mathematics 2
- ESAT Physics

The project is built around timed practice, mistake analysis, weekly review and targeted repair.

## File map

| File | Role |
|---|---|
| `CLAUDE.md` | Source of truth for working rules and project standards |
| `schedule.md` | 12-week operational schedule |
| `roadmap.md` | Phase-level milestones and readiness gates |
| `strategy.md` | Exam technique and pacing rules only |
| `prep.md` | Reference notes on ESAT format, sources and material selection |
| `error_log.md` | Question-level mistake log |
| `score_dashboard.md` | Weekly score and timing tracker |
| `CHANGELOG.md` | Chronological record of material project changes |

## Folder structure

| Folder | Use |
|---|---|
| `papers/esat` | Official ESAT specimens, sample tests, mark schemes and guidance |
| `papers/nsaa` | NSAA papers and mark schemes |
| `papers/engaa` | ENGAA papers and mark schemes |
| `papers/tmua` | TMUA papers and mark schemes |
| `papers/pat` | Original PAT papers and mark schemes only |
| `papers/uploaded_docs` | Uploaded MCQ documents |
| `mocks` | Constructed timed sets |
| `reviews` | Corrections, weak-topic repair and post-test analysis |

Do not create separate answer folders. Keep each paper, mark scheme, official solution and examiner report in the same exam folder.

## Naming convention

Use:

```text
EXAM_YEAR_type.ext
```

Examples:

```text
ENGAA_2023_paper.pdf
ENGAA_2023_mark_scheme.pdf
NSAA_2022_paper.pdf
NSAA_2022_mark_scheme.pdf
PAT_2021_paper.pdf
PAT_2021_mark_scheme.pdf
TMUA_2020_paper_1.pdf
TMUA_2020_paper_1_mark_scheme.pdf
```

## Basic workflow

1. Complete timed set.
2. Mark answers.
3. Record score, timing and skipped questions in `score_dashboard.md`.
4. Add every missed or lucky question to `error_log.md`.
5. Identify recurring patterns.
6. Adjust the next week based on evidence, not mood.

## Review principle

Do not simply record scores. The objective is to identify repeatable failure patterns and remove them.
