# technique.md

# Master technique cross-reference for question-by-question strategy files

## Universal route test
Units → substitute → option test → proportionality/scaling → graph area/gradient → limiting case → estimate → derive only if short.

## Maths
- §3.1 Substitution and special values: use x=0, 1, -1, 2, or test answer options before expansion.
- §3.2 Quadratics/functions: discriminant for intersections/tangency; complete square for minima/maxima.
- §3.3 Graphs and transformations: track vertex, key point, intercept, asymptote or number of intersections.
- §3.4 Ratios, percentages, scaling: multipliers; convenient totals; k, k², k³ scaling.
- §3.5 Probability: complement, tables, trees only when dependent, outcome counting.
- §3.6 Geometry/trig: redraw; mark invariants; exact angles; sign/interval counting.
- §3.7 Binomial/sequences/series: write first terms; target required coefficient; check convergence.

## Physics
- §4.1 Units/dimensional analysis: check units first; build unknown formulae from dimensions.
- §4.2 Graph meaning: area, gradient, intercept or trend before formulas.
- §4.3 Energy/work/power: conservation and input/useful/wasted energy audit.
- §4.4 Mechanics/FBD/moments: draw forces, choose axes/pivot, use resultant/equilibrium.
- §4.5 Momentum/impulse: choose sign; conserve momentum; impulse = Δp.
- §4.6 Circuits/electricity: reduce circuit; current first; transformer power conservation.
- §4.7 Waves/sound/optics: v=fλ; particle travel 4A/cycle; echo is there-and-back; draw rays.
- §4.8 Nuclear/half-life: A/Z ledger; subtract background before halving.
- §4.9 Thermal/gases: pV, conduction gradients, radiation surface rules, limiting cases.
