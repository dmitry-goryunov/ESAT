# TMUA Paper 1 Strategy, relevant sections for ESAT Maths

Scope reviewed: TMUA Paper 1 files available in the project folder, 2016 to 2023, plus specimen material.  
Use TMUA Paper 1 as a Maths fluency and shortcut drill. It is harder and more abstract than ESAT Maths in places, so use selected questions rather than treating it as the main ESAT benchmark.

---

## 1. Format and timing

- Paper 1: 75 minutes.
- 20 multiple-choice questions.
- No calculator.
- No formula booklet.
- No penalty for wrong answers.

Raw TMUA timing:

| Block | Questions | Time | Time per question |
|---|---:|---:|---:|
| Full paper | 20 | 75 min | 3 min 45 sec |

ESAT-style use:

| Drill | Questions | Time | Aim |
|---|---:|---:|---|
| Fast fluency set | 10 | 25 to 30 min | shortcut recognition |
| Hard set | 10 | 35 to 40 min | deeper algebra/graph/trig |
| Full TMUA paper | 20 | 70 to 75 min | endurance, not ESAT timing |

For ESAT, do not let TMUA distort pacing expectations. ESAT is faster and usually less proof-heavy.

---

## 2. What to use from TMUA

High relevance:
- Algebraic manipulation.
- Quadratics and discriminants.
- Graph intersections and number of solutions.
- Functions and transformations.
- Sequences and series.
- Trigonometry.
- Coordinate geometry.
- Binomial and coefficient targeting.
- Probability and counting.
- Special values and substitution.

Lower priority:
- Heavy calculus beyond ESAT.
- Long proof-style inequalities.
- Abstract function/integral questions unless used as challenge drills.

## 3. TMUA quick-technique map

| Question type | Fast technique |
|---|---|
| Algebraic identity | Substitute x = 0, 1, -1, 2 |
| Equation with options | Substitute options |
| Quadratic/root condition | Discriminant, sum/product, completing square |
| Number of solutions | Sketch or sign chart |
| Trig equation | Factor, use identities, count range carefully |
| Sequence/series | Write first terms, look for telescoping |
| Binomial coefficient | Target only required term |
| Function transformation | Track minimum point or one key point |
| Coordinate geometry | Use distance/vector/gradient before full expansion |
| Probability | Count outcomes systematically |
| Graph area | Sketch first; integrate only if needed |
| Max/min expression | Substitute a bounded variable, e.g. u = sin x |

## 4. Year-by-year strategy notes

### TMUA 2016 Paper 1

Pattern:
- Excellent baseline for algebra, graph counting, trig, sequences and geometry.
- Several questions are solved faster by pattern recognition rather than full derivation.

Best techniques:
- Generate first few terms for sequence questions.
- Use graph intersections for number of solutions.
- Use stationary points or graph shape rather than solving high-degree equations.
- Use substitution for algebraic identities.
- For geometry, draw clean diagrams and introduce one variable.

Priority technique questions:
- Q4, Q6, Q7, Q8, Q10, Q13, Q14, Q17, Q18, Q20.

---

### TMUA 2017 Paper 1

Pattern:
- More algebraic and functions-heavy.
- Good practice for avoiding unnecessary expansion.
- Useful for logarithms, quadratics, transformations, trigonometric identities and coordinate geometry.

Best techniques:
- Factor before expanding.
- Use discriminant for “no intersection” or “two roots”.
- Use special values for algebraic equivalence.
- Use graph shape for function transformations.
- Use sign/range checks before algebra.

Priority:
- Algebra, functions, trig and coordinate geometry questions.
- Skip calculus-heavy questions on first ESAT pass.

---

### TMUA 2018 Paper 1

Pattern:
- Strong mixed Maths paper.
- Good for sequences, combinatorics, functions, graphs, calculus-light optimisation and geometry.

Best techniques:
- Use first few terms for recurrence/series.
- For binomial/combinatorics, target the requested coefficient only.
- For functions, locate turning point or key intercept first.
- For coordinate geometry, use symmetry if present.
- For integration/area, sketch before calculating.

Priority:
- Use as a second-stage Maths drill after NSAA/ENGAA basics.

---

### TMUA 2019 Paper 1

Pattern:
- Good ESAT-relevant algebra and graph reasoning.
- Strong on transformations, exact trig, logarithms and inequalities.

Best techniques:
- Use answer options to reduce algebra.
- Complete square for quadratic transformations.
- For trig, convert to one trig function when possible.
- For logs, check domains first.
- For inequalities, use factor/sign table.

Priority:
- High value for ESAT Maths 2 style fluency.

---

### TMUA 2020 Paper 1

Pattern:
- Compact, abstract, and time-consuming.
- Good for high-level technique, less direct as an ESAT benchmark.

Best techniques:
- Do not expand unless necessary.
- Use symmetry and invariants.
- For functions, track transformations.
- For equations, use substitution and range.
- For probability, define sample space carefully.

Priority:
- Use selected questions only under ESAT preparation.
- Treat hard questions as review material, not timing calibration.

---

### TMUA 2021 Paper 1

Pattern:
- Good mix of algebra, functions, trigonometry, coordinate geometry and sequences.
- Several questions reward graph interpretation over algebra.

Best techniques:
- Graph first for solution counting.
- For transformations, track minimum/maximum points.
- Use discriminant for tangency/intersection.
- Use special values for identities.
- Use sign charts for inequalities.

Priority:
- High-value Paper 1 for ESAT Maths fluency.

---

### TMUA 2022 Paper 1

Pattern:
- Demanding but useful.
- Good for exact algebra, trig, functions, graph reasoning and sequences.

Best techniques:
- Reduce to a simpler variable when expressions are complicated.
- Use endpoint and range checks.
- For trig, count solutions carefully across the interval.
- For circles/coordinate geometry, use centre and radius first.
- For binomial/series, avoid full expansion.

Priority:
- Use after building speed on NSAA/ENGAA.

---

### TMUA 2023 Paper 1

Pattern:
- Highly abstract, less like ESAT but excellent for top-end technique.
- Topics include integrals, discriminants, series, trigonometry, graph transformations, circle geometry, roots of cubics, max/min, coordinate geometry and probability.

Best techniques:
- For integrals with parameters, treat integrals as linear equations.
- For “graphs do not meet”, use discriminant.
- For trigonometric solution counts, factor and count the transformed angle range.
- For circle distance, maximum distance = distance between centres + radii where appropriate.
- For `f(x)` transformed minimums, track the minimum point, not the whole graph.
- For maximising expressions in sin x, set `u = sin x` and optimise over `-1 ≤ u ≤ 1`.
- For probability from finite integer choices, enumerate valid cases.

Priority questions:
- Q2, Q5, Q7, Q8, Q9, Q11, Q12, Q13, Q14, Q16, Q17, Q18, Q20.
- Q1, Q3, Q4, Q10, Q19 are useful but less direct for ESAT.

## 5. Best TMUA strategies

### Strategy 1: Substitute before expanding

TMUA loves expressions that look algebra-heavy but collapse under a test value.

Use:
- x = 0
- x = 1
- x = -1
- x = 2
- n = 1, 2
- θ = 0°, 30°, 45°, 60°, 90° where legal

### Strategy 2: Use the discriminant

For line/quadratic, tangent/no intersection/two intersections:

`discriminant < 0`, `= 0`, or `> 0`.

This is often faster than solving roots.

### Strategy 3: Count solutions by graph

For trig, tan, rational or high-degree equations:
- sketch each side
- identify intervals
- count intersections
- do not solve every root

### Strategy 4: Target the requested term

For binomial expansions:
- find the general term
- set exponent equal to what is required
- compute only that term

### Strategy 5: Track transformations via key points

For functions:
- minimum point
- maximum point
- intercepts
- asymptotes
- domain/range

Do not transform the whole curve unless necessary.
