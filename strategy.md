# ESAT Strategy

## Core principle

ESAT rewards fast correct decisions under pressure. Do not aim for elegant written solutions during timed work.

## Pacing rules

| Situation | Action |
|---|---|
| Immediate route visible | Solve directly |
| Route visible but algebra looks long | Estimate or eliminate first |
| No clear route after 45 to 60 seconds | Eliminate, guess and move on |
| Two answers left | Use units, limiting cases, substitution or graph shape |
| Question looks unusually long | Skip and return only if time remains |

## Skipping rule

If stuck for more than 60 seconds:
1. eliminate impossible answers
2. make the best available guess
3. mark for return if appropriate
4. move on

One question must not destroy the module.

## Fast methods

Prefer:
- dimensional checks
- limiting cases
- proportional reasoning
- symmetry
- graph-shape recognition
- rough magnitude checks
- substituting answer choices
- spotting invariant quantities

Avoid:
- full derivation unless short
- long algebra when elimination works
- checking every answer after the correct one is clear
- re-solving the same question twice during the first pass

## First pass / second pass method

| Pass | Goal |
|---|---|
| First pass | collect all quick marks and avoid traps |
| Second pass | return to marked questions with a clear method |
| Final 2 minutes | fill all blanks, prioritise questions with only two options left |

## Review rule

After every timed set, classify each error as one primary type:
- knowledge gap
- algebra slip
- careless arithmetic
- timing panic
- trap/misread
- overthinking
- stamina/fatigue
- method selection

The pattern matters more than the single question.
